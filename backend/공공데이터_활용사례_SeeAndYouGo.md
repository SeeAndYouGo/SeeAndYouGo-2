# SeeAndYouGo 공공데이터 활용사례 문서

## 📋 프로젝트 개요

**프로젝트명**: SeeAndYouGo  
**서비스 도메인**: www.seeandyougo.com  
**개발 목적**: 충남대학교 학생식당 정보 제공  
**활용 공공데이터**: 충남대학교 정보화본부 API (WiFi 연결 정보, 학생식당 메뉴 정보)

## 🔗 공공데이터 활용 현황

### 1. 활용 API 정보
#### 1.1 WiFi 연결 정보 API
- **데이터 내용**: 각 학생식당별 실시간 WiFi 접속자 수
- **활용 목적**: 학생식당별 실시간 혼잡도 추정 및 제공
- **갱신 주기**: 5분마다 업데이트
- **대상 식당**: 제1학생회관, 제2학생회관, 제3학생회관, 상록회관(제4학생회관), 생활과학대

#### 1.2 학생식당 메뉴 정보 API  
- **API 엔드포인트**: `https://api.cnu.ac.kr/svc/offcam/pub/FoodInfo`
- **데이터 내용**: 주간 식단표, 메뉴명, 가격정보, 학생/교직원 구분
- **활용 목적**: 실시간 식단 정보 제공 및 메뉴 추천
- **갱신 주기**: 주간 단위 업데이트
- **제공 범위**: 조식, 중식, 석식 전 메뉴 정보

## 🎯 주요 서비스 기능

### 3.1 실시간 혼잡도 서비스
**기능 설명**: WiFi 접속자 수 기반 학생식당별 혼잡도 실시간 제공
- **데이터 활용**: 정보화본부 WiFi 연결 정보 API
- **사용자 혜택**: 대기시간 최소화, 효율적인 식당 이용 계획 수립
- **업데이트 주기**: 5분마다 자동 갱신
- **표시 방식**: 연결된 기기 수를 기반으로 한 혼잡도 레벨 제공

### 3.2 통합 식단 정보 서비스
**기능 설명**: 6개 학생식당의 실시간 메뉴 정보 통합 제공
- **데이터 활용**: 정보화본부 학생식당 메뉴 정보 API
- **제공 정보**: 
  - 일간/주간 식단표
  - 학생식당/교직원식당 구분 메뉴
  - 메뉴별 가격 정보
  - 신메뉴 하이라이트 기능
- **사용자 혜택**: 한 번의 접속으로 모든 식당 메뉴 비교 가능

### 3.3 메뉴 리뷰 및 평가 시스템
**기능 설명**: 사용자 경험 기반 메뉴 품질 평가 시스템
- **주요 기능**:
  - 5점 척도 메뉴 평가
  - 사진 첨부 리뷰 작성
  - 실시간 평점 집계 및 표시
  - 인기 메뉴 랭킹 시스템
- **데이터 연계**: 공공데이터 메뉴 정보와 사용자 리뷰 데이터 결합
- **사용자 혜택**: 메뉴 선택 시 참고할 수 있는 신뢰성 있는 후기 정보

## 📊 사용자 가치 및 사회적 효과

### 4.1 학생 생활 편의성 향상
- **시간 절약**: 사전 메뉴 확인으로 식당 선택 시간 단축
- **대기시간 최소화**: 혼잡도 정보 활용한 효율적인 식사 계획
- **경제적 효율**: 가격 정보 제공으로 합리적인 식사 선택 지원

### 4.2 식당 운영 효율성 기여
- **수요 분산**: 혼잡도 정보 제공으로 자연스러운 이용객 분산 효과
- **품질 관리**: 리뷰 시스템을 통한 메뉴 품질 피드백 제공
- **데이터 기반 운영**: 이용 패턴 분석을 통한 운영 최적화 지원

## 🎯 기대 효과 및 확장 계획

### 6.1 정량적 기대효과
- **사용자 만족도 향상**: 식당 선택 시간 50% 단축 예상
- **혼잡 분산 효과**: 피크타임 대기시간 30% 감소 목표
- **정보 접근성**: 6개 식당 정보를 하나의 플랫폼에서 통합 제공

### 6.2 향후 확장 계획
- **AI 기반 서비스**: 개인별 메뉴 추천 알고리즘 고도화
- **모바일 최적화**: PWA 기반 모바일 앱 서비스 확장
- **다른 대학 확산**: 타 대학 공공데이터 API 연계 확대
