```text

```

| 판단 항목 | 산출 근거 | 적용 값 |

| 지난주 같은 요일 같은 시간대 값 | 전체 조합 비교 시 가장 높은 설명력 | 90.9% |
| 최근 8주 같은 요일 같은 시간대 평균 | 지난주 값 보정용으로만 소폭 개선 | 9.1% |
| 같은 요일 같은 시간대 전체 평균 | 최적 조합에서 기여 없음 | 0% |
| 같은 메뉴 이력 평균 | 테스트 구간 기준 기본 공식 개선 없음 | 0% |
