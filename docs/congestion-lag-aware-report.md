# 혼잡도 지연 반응 개선 요약

- 기준 기간: `2025-01-01 ~ 2025-12-31`
- 검증 기간: `2026-01-01 ~ 2026-03-27`
- 목적: `30분 후`, `60분 후` 예측이 피크 직후에 늦게 반응하는 문제를 줄일 수 있는지 확인

## 핵심 결론

- `5분 후`는 `현재값 유지` 대신 `현재값 + 최근 변화량 10%`를 적용해도 무리가 없었습니다.
- `30분 후`, `60분 후`는 `피크 시간대 인식 + 최근 추세 반영`을 넣으면 전반적으로 조금 더 빨리 반응했습니다.
- 다만 모든 식당에 동일하게 적용하는 것은 적절하지 않았습니다.
- `생활과학대 60분`, `학생생활관 30분/60분`은 기존 규칙 유지가 더 적절했습니다.

## 전체 결과

| 구간 | 기존 MAE | 개선 MAE | 기존 감소 구간 MAE | 개선 감소 구간 MAE |
|---|---:|---:|---:|---:|
| 5분 후 | 3.6 | 3.6 | 5.9 | 5.8 |
| 30분 후 | 11.1 | 10.8 | 11.4 | 11.1 |
| 60분 후 | 16.1 | 15.6 | 14.5 | 13.2 |
