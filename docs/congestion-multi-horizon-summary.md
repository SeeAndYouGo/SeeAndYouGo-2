# 혼잡도 예측 범위별 단순 운영안

작성일: 2026-03-28
검증 구간: 2026-03-01 ~ 2026-03-27
검증 단위: 식당별 5분 슬롯

## 1. 결론

하나의 단순 규칙으로 모든 예측 구간을 처리하는 것은 적절하지 않았습니다. 예측 범위에 따라 가장 적합한 단순 규칙이 달랐습니다.

권장안은 아래와 같습니다.

| 예측 범위 | 권장 규칙 |
|---|---|
| 5분 후 | 현재값 유지 |
| 30분 후 | 목표 시점 기준선 + 현재 편차 80% 반영 |
| 60분 후 | 목표 시점 기준선 + 현재 편차 70% 반영 |

## 2. 기준 정의

### 현재값

현재 시점의 실제 혼잡도 값입니다.

### 현재 기준선

현재 시점 슬롯에 대해, 개장 전 기준으로 계산한 평소 예상값입니다.

```text
현재 기준선 = 0.909 × 지난주 같은 요일 같은 시간대 값
          + 0.091 × 최근 8주 같은 요일 같은 시간대 평균
```

### 목표 시점 기준선

예측하려는 목표 시간대 슬롯에 대해, 개장 전 기준으로 계산한 평소 예상값입니다.

```text
목표 시점 기준선 = 0.909 × 지난주 같은 요일 같은 시간대 값
              + 0.091 × 최근 8주 같은 요일 같은 시간대 평균
```

### 현재 편차

현재 실제 혼잡도가 평소 기준선보다 얼마나 높은지 또는 낮은지를 의미합니다.

```text
현재 편차 = 현재값 - 현재 기준선
```

## 3. 권장 공식

### 5분 후

```text
예측값 = 현재값
```

### 30분 후

```text
예측값 = 목표 시점 기준선 + 0.8 × 현재 편차
```

### 60분 후

```text
예측값 = 목표 시점 기준선 + 0.7 × 현재 편차
```

## 4. 비교한 단순 방식

| 방식 | 설명 |
|---|---|
| 현재값 유지 | 현재 실측값을 그대로 미래값으로 사용 |
| 최근 15분 평균 | 현재 포함 최근 3개 슬롯 평균 사용 |
| 목표 시점 기준선 | 목표 시점의 평소값만 사용 |
| 편차 반영 방식 | 목표 시점 기준선에 현재 편차를 일부 반영 |

## 6. 왜 이렇게 판단하였는지

| 예측 범위 | 핵심 판단 |
|---|---|
| 5분 후 | 현재 상태가 거의 그대로 이어지는 구간이므로 현재값이 가장 강함 |
| 30분 후 | 현재 상태를 그대로 유지한다고 보기 어렵기 때문에 목표 시간대의 평소값이 필요함 |
| 60분 후 | 멀어질수록 현재 편차는 점차 약해지고, 목표 시간대의 구조적 기준선이 더 중요해짐 |

## 7. 최종 운영안

| 단계 | 적용 방식 |
|---|---|
| 개장 전 | 모든 슬롯에 대해 목표 시점 기준선 미리 계산 |
| 영업 중 5분 후 | 현재값 사용 |
| 영업 중 30분 후 | 목표 시점 기준선 + 현재 편차 80% |
| 영업 중 60분 후 | 목표 시점 기준선 + 현재 편차 70% |
| 예외 상황 | 현재값이 없으면 목표 시점 기준선 사용 |

## 8. 팀 공유용 한 줄 정리

| 항목 | 요약 |
|---|---|
| 5분 후 | 지금 값이 가장 중요합니다. |
| 30분 후 | 지금 값만 복사하지 말고, 목표 시간대의 평소값에 현재 편차를 얹는 편이 좋습니다. |
| 60분 후 | 시간이 멀어질수록 목표 시간대 기준선이 더 중요해지며, 현재 편차는 조금만 반영하는 편이 좋습니다. |
