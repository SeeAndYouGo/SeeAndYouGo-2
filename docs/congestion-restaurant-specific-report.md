# 식당별 혼잡도 예측 통합 보고서

작성일: 2026-03-28  
검증 구간: 2026-03-01 ~ 2026-03-27  
기준 단위: 식당별 5분 데이터

이 문서는 식당별 실제값과 예측값을 날짜 기준으로 비교한 그래프를 한 곳에 모아둔 보고서입니다. 이번에는 모든 식당에 같은 규칙을 적용하지 않고, 식당 특성에 따라 예측 방식을 다르게 적용하였습니다.

평소값은 아래 기준으로 계산하였습니다.

`평소값 = 지난주 같은 요일 같은 시간대 값 90.9% + 최근 8주 같은 요일 같은 시간대 평균 9.1%`

## 1. 한눈에 보는 결론

공통 규칙 하나로도 운영은 가능했지만, 식당별 규칙을 따로 두었을 때 `30분 후`, `60분 후` 예측이 조금 더 좋아졌습니다.

전체 기준 결과는 아래와 같습니다.

| 예측 범위 | 식당별 규칙 적용 후 평균 차이 | `±10명` 이내 비율 |
|---|---:|---:|
| 5분 후 | `4.5명` | `87.4%` |
| 30분 후 | `11.5명` | `71.9%` |
| 60분 후 | `15.2명` | `66.2%` |

특히 `학생생활관`, `제1학생회관`, `제3학생회관`은 식당별 규칙을 두는 편이 더 자연스러웠습니다.

## 2. 식당별 추천 방식

| 식당 | 특성 요약 | 5분 후 | 30분 후 | 60분 후 |
|---|---|---|---|---|
| 상록회관 | 비교적 안정적이고 점심이 저녁보다 조금 높음 | 지금 값 그대로 | 30분 뒤 평소값 + 현재 차이 80% | 60분 뒤 평소값 + 현재 차이 60% |
| 생활과학대 | 0명 구간이 많고 검증 구간에서는 변화가 거의 없음 | 지금 값 그대로 | 지금 값 그대로 | 지금 값 그대로 |
| 제1학생회관 | 가장 규모가 크고 변동도 큼 | 지금 값 그대로 | 30분 뒤 평소값 + 현재 차이 90% | 60분 뒤 평소값 + 현재 차이 80% |
| 제2학생회관 | 점심 피크가 뚜렷한 중간 규모 식당 | 지금 값 그대로 | 30분 뒤 평소값 + 현재 차이 80% | 60분 뒤 평소값 + 현재 차이 60% |
| 제3학생회관 | 규모는 작지만 점심 집중이 분명함 | 지금 값 그대로 | 30분 뒤 평소값 + 현재 차이 70% | 60분 뒤 평소값 + 현재 차이 50% |
| 학생생활관 | 저녁 비중이 높고 0명 구간도 많음 | 지금 값 그대로 | 지금 값 그대로 | 60분 뒤 평소값 + 현재 차이 30% |

## 6. 최종 정리

이번 결과를 기준으로 보면, `5분 후`는 대부분 식당에서 지금 값을 그대로 쓰는 것이 가장 단순하고 잘 맞았습니다. 다만 `30분 후`, `60분 후`는 식당마다 현재 흐름을 얼마나 오래 유지하는지가 달랐습니다. 따라서 운영에서는 모든 식당에 같은 규칙을 쓰기보다, 위 표처럼 식당별 반영 비율을 다르게 두는 편이 더 적절하다고 판단하였습니다.
