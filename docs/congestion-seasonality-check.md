# 시즌성 기준 평균값 비교

- 기준 기간: `2025-01-01 ~ 2025-12-31`
- 검증 기간: `2026-01-01 ~ 2026-03-27`
- 비교 목적: 평균값을 `전 기간 같은 요일 같은 시간대`로 볼지, `시즌을 나눈 같은 요일 같은 시간대`로 볼지 확인

## 현재 기준

현재 코드 기준 평균값은 `같은 식당 + 같은 요일 + 같은 시간대`의 과거 평균입니다. 시즌 구분은 들어가 있지 않습니다.

## 이번 비교에서 쓴 시즌 구분

- `1~2월`: 겨울 비수기
- `3~6월`: 1학기
- `7~8월`: 여름 비수기
- `9~12월`: 2학기

시즌 기준 평균값은 `같은 식당 + 같은 시즌 + 같은 요일 + 같은 시간대` 평균을 우선 사용하고, 데이터가 부족하면 기존 전 기간 평균으로 되돌아가도록 했습니다.

## 전체 결과

| 비교 항목 | 전 기간 기준 MAE | 시즌 구분 기준 MAE | 전 기간 감소 구간 MAE | 시즌 구분 감소 구간 MAE |
|---|---:|---:|---:|---:|
| 30분 평균값만 사용 | 31.1 | 25.8 | 34.6 | 29.5 |
| 60분 평균값만 사용 | 31.9 | 26.4 | 31.9 | 26.8 |
| 30분 기존 규칙 포함 | 10.8 | 10.6 | 11.1 | 11.5 |
| 60분 기존 규칙 포함 | 15.6 | 15.1 | 13.2 | 14.0 |

## 해석

- 이 결과를 기준으로, 시즌 구분을 넣는 편이 실제로 더 나은지 판단할 수 있습니다.
- 만약 시즌 구분 기준이 더 좋다면, 평균값 계산 기준은 전 기간이 아니라 학기/방학 구분을 우선 사용하는 방향이 맞습니다.
